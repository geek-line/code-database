About Code Database
Code Databaseの紹介とその特徴について説明します。
Code Databaseとは？
Code Databaseは、エンジニアを目指している人たちへのプログラミングに関する情報を発信しています。JavaScriptのようなWEB系の言語からC++のような組み込み系の言語まで幅広く紹介していきます。フレームワークやライブラリのインストールと環境統合から基本的な文法知識、実践的なアプリの開発まで用途に応じた記事を作成しています。
Code Databaseの特徴
1. 多くの言語・ライブラリに関する記事がある

Code DatabaseにはJavaScript, Vue.js, React.js, Nuxt.js, Python, Go言語, Unity, C++, OpenCVなどあらゆるプログラミング言語やフレームワークの情報を発信しています。

2. ハンズオン形式で学べる記事がある

GolangとVue.jsとChart.jsで小さなSPAアプリを作る方法について解説の記事のように実際に学びながらアプリを作成することができます。このようなハンズオン形式で学べる記事をどんどんアップする予定です。
